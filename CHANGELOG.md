## [1.0.0] - 2026-01-06

### Added
- Initial release of UX Audit Agent Plugin for Claude Code
- Three cognitive UX analysis skills:
  - `ux-review`: Quick post-development cognitive UX review
  - `ux-audit`: Deep cognitive UX audit with persona-based analysis
  - `user-expectations`: Analyze implicit expectations created by interface elements
- UX Audit Agent for autonomous evidence-based audits with scoring (0-100)
- Comprehensive UX framework based on:
  - Nielsen Norman Usability Heuristics
  - Ben Shneiderman's Eight Golden Rules
  - Don Norman's Cognitive Design Principles
- Stroop Effect analysis for visual-semantic conflicts
- Visual hierarchy auditing
- Persona-based evaluation system
- Severity classification system (Critical to Low)

### Features
- Cognitive Gap Detection identifying friction points
- Color-action mismatch detection
- Platform convention validation
- Domain-specific semantic analysis
- Evidence-based scoring and prioritization
- Support for React, Vue, Svelte, Angular, HTML, CSS, Tailwind
